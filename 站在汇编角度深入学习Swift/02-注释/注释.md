# 注释

## 单行注释

``` swift
//	我是一条注释语句
```

## 多行注释

``` swift

/*
	我
	是
	一
	条
	多
	行
	注
	释
*/

```

## swift支持嵌套注释

``` swift

/* 我
	 是
	 外
	 层
	 注
	 释
	/*
		我
		是
		内
		层
		注
		释
	*/
*/

```
